## üéì Learning Paths

### Path 1: "I Want to Use This Immediately"
1. Read: [QUICK_START.md](QUICK_START.md) (5 min)
2. Copy: `components/gantt/` and `lib/gantt/` to your project
3. Install: Dependencies from quick start
4. Code: Use example from [basic-gantt.tsx](app/examples/basic-gantt.tsx)
5. Deploy: Follow [DEPLOYMENT_CHECKLIST.md](DEPLOYMENT_CHECKLIST.md)

**Time: 30 minutes**

### Path 2: "I Want to Understand Everything"
1. Read: [INDEX.md](INDEX.md) (5 min)
2. Explore: [/playground](/playground) (15 min)
3. Study: [ARCHITECTURE.md](ARCHITECTURE.md) (30 min)
4. Reference: [README.md](README.md) (20 min)
5. Code: Review [components/gantt/](components/gantt/) source (30 min)

**Time: 100 minutes**

### Path 3: "I Want to Publish as Open Source"
1. Setup: [GITHUB_SETUP.md](GITHUB_SETUP.md) (30 min)
2. Review: [DEPLOYMENT_CHECKLIST.md](DEPLOYMENT_CHECKLIST.md) (20 min)
3. Deploy: Follow deployment section (15 min)
4. Publish: npm publishing steps (10 min)
5. Announce: Share with community (10 min)

**Time: 85 minutes**

### Path 4: "I Want to Extend/Customize"
1. Learn: [ARCHITECTURE.md](ARCHITECTURE.md) (30 min)
2. Review: Component source in [components/gantt/](components/gantt/) (30 min)
3. Study: Type definitions in [lib/gantt/types.ts](lib/gantt/types.ts) (10 min)
4. Implement: Create custom components (60+ min)
5. Reference: [README.md](README.md) APIs during coding (as needed)

**Time: 140+ minutes (depends on customization)**
